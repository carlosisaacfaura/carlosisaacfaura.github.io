SpaceX Landing Prediction

This project focuses on predicting the success of the landing of the Falcon 9 rocket's first stage using advanced data analysis techniques, machine learning, and data visualizations. Using the SpaceX API and records from Wikipedia, models such as Logistic Regression, SVM, Decision Tree, and KNN were employed. The visualizations highlighted the preference for the CCAFS LC-40 launch site and the high success rate of the KSC LC-39A site, facilitating the interpretation of complex results.

Discounted Sales Analysis

This project offers an in-depth analysis of the profitability impacts of discounts across various product categories and segments using the SuperStore Dataset. Leveraging pre-processed data, the visuals provide insights into the relationship between discount levels and profitability. The analysis uncovers key patterns, such as the diminishing profitability with increasing discount percentages and highlights the most profitable segments and sub-categories. The work serves as a guide for effective discounting strategies, emphasizing data-driven decision-making. Data preprocessing was not required as the data was already preprocessed.
