Assignment: Machine Learning Prediction¶

Estimated time needed: 60 minutes

Space X advertises Falcon 9 rocket launches on its website with a cost of 62 million dollars; other providers cost upward of 165 million dollars each, much of the savings is because Space X can reuse the first stage. Therefore if we can determine if the first stage will land, we can determine the cost of a launch. This information can be used if an alternate company wants to bid against space X for a rocket launch. In this lab, you will create a machine learning pipeline to predict if the first stage will land given the data from the preceding labs.

In [ ]:
def plot_confusion_matrix(y,y_predict):
    "this function plots the confusion matrix"
    from sklearn.metrics import confusion_matrix

    cm = confusion_matrix(y, y_predict)
    ax= plt.subplot()
    sns.heatmap(cm, annot=True, ax = ax); #annot=True to annotate cells
    ax.set_xlabel('Predicted labels')
    ax.set_ylabel('True labels')
    ax.set_title('Confusion Matrix'); 
    ax.xaxis.set_ticklabels(['did not land', 'land']); ax.yaxis.set_ticklabels(['did not land', 'landed']) 
    plt.show()

import matplotlib.pyplot as plt

# Create a list of the models and their corresponding accuracy scores
models = ['Logistic Regression', 'SVM', 'Decision Tree', 'KNN']
accuracy_scores = [lr_accuracy,svm_accuracy,tree_accuracy, knn_accuracy]

# Create a bar chart to visualize the model accuracies
plt.figure(figsize=(10, 6))
bars = plt.bar(models, accuracy_scores, color=['lightblue', 'pink', 'lightgreen', 'lightgrey'])

for bar in bars:
    yval = bar.get_height()
    plt.text(bar.get_x() + bar.get_width()/2, yval + 0.01, round(yval*100, 2), ha='center', va='bottom')

plt.xlabel('Classification Models')
plt.ylabel('Accuracy')
plt.title('Accuracy Comparison of Different Classification Models')


plt.show()
